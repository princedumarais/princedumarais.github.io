Install Prestashop
==================

- Install Git, Ansible, Vagrant, Virtualbox
- $ vagrant up

Go to admin and disable the following modules :
- Banner block
- Image slider for your home page
- Theme configurator

Remove demo data
- intall database cleaner and run it
